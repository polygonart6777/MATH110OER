Hint 3.1.5.1.

See Hint 3.1.1.3 for the definition of the transpose and Hint 3.1.2.1 for the definition of matrix multiplication.

Look back into [4], Chapter 2, Theorem 2.1.1, for some properties of matrix operation.

A square matrix \(A\) is called symmetric if \(A=A^T\text{.}\)

Two square matrices \(A,B\) of the same size are said to commute if \(AB=BA\text{.}\)
