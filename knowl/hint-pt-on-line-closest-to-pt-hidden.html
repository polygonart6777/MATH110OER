If a line \(\ell\) is given by the vector equation
\begin{equation*} \mbf{x} = \mbf{a} + t\mbf{d} \end{equation*}
and \(\mbf{p}\) is the position vector of a point \(P\text{,}\) then
\begin{equation*} \mbf{q} := \mbf{a} + \operatorname{proj}_{\mathbf{d}}(\mathbf{p}-\mathbf{a}), \end{equation*}
is the position vector of the point \(Q\) on \(\ell\) which is closest to the point \(P\text{.}\)
