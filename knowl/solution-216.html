Solution 4.2.11.c.1.
We know that the matrix is invertible if \(det(A) \ne 0 \text{,}\) therefore we can compute the determinant in order to identify the values of \(c \) which make the determinant non-zero. We choose to expand along the first row because the zero in the third column of the first row will simplify our cofactor calculation.
\begin{align*} det(A) = c (-1)^{1+1}\begin{vmatrix}2 \amp c \\ c \amp 5 \end{vmatrix} + 1 (-1)^{1+2}\begin{vmatrix} 0 \amp c \\ -1 \amp 5 \end{vmatrix} \amp \ne 0\\ c(10-c^2) -(0-c) \amp \ne 0\\ c(9-c^2) \amp \ne 0\\ c \amp \ne 0,-3,3 \end{align*}
The matrix will be invertile if \(c\) is not equal to 0, -3 or 3.
