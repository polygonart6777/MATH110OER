Solution 4.2.11.d.1.
We know that the matrix is invertible if \(det(A) \ne 0 \text{,}\) therefore we can compute the determinant in order to identify the values of \(c \) which make the determinant non-zero. We choose to expand along the first column this time just to mix things up.
\begin{align*} det(A) = 4 (-1)^{1+1}\begin{vmatrix}2 \amp c \\ c \amp 4 \end{vmatrix} + c(-1)^{2+1}\begin{vmatrix} c \amp 3 \\ c \amp 4 \end{vmatrix}+ 5(-1)^{3+1}\begin{vmatrix} c \amp 3 \\ 2 \amp c \end{vmatrix} \amp \ne 0\\ 4(8-c^2) -(4c-3c) +5(c^2-6) \amp \ne 0\\ 2 \amp \ne 0 \end{align*}
