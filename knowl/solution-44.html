Solution 1.3.5.1.
According to Hint 1.3.5.1, we only need to find a point \(A\) on the plane and a normal vector \(\mbf{n}\) to the plane.
  1. For the plane with general equation \(5x_1+x_2+x_3 =1\text{,}\) we can choose
    \begin{equation*} \mbf{n} = \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} \text{ and } A = (1, -4, 0). \end{equation*}
    We then compute
    \begin{equation*} \mbf{v} := \vect{OP}-\vect{OA} = \begin{bmatrix} 2\\ 3\\ 0 \end{bmatrix} - \begin{bmatrix} 1\\ -4\\ 0 \end{bmatrix} = \begin{bmatrix} 1\\ 7\\ 0 \end{bmatrix}, \end{equation*}
    so that
    \begin{equation*} \mbf{n} \cdot \mbf{v} = \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} \cdot \begin{bmatrix} 1\\ 7\\ 0 \end{bmatrix} = 5+7 = 12 \end{equation*}
    and
    \begin{equation*} \mbf{n} \cdot \mbf{n} = \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} \cdot \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} = 25+1+1 = 27. \end{equation*}
    We can now compute (using Hint 1.2.8.1)
    \begin{equation*} \operatorname{proj}_{\mathbf{n}}(\mathbf{v}) = \left( \frac{\mbf{n}\cdot \mbf{v}}{\mbf{n}\cdot \mbf{n}} \right) \mbf{n} = \frac{12}{27} \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} = \frac{4}{9} \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} \end{equation*}
    and thus
    \begin{align*} \operatorname{dist} (\mathcal{P}, P) \amp = \norm{\operatorname{proj}_{\mathbf{n}}\mathbf{v}}\\ \amp = \sqrt{ \left(\frac{20}{9}\right)^2 + \left(\frac{4}{9}\right)^2 + \left(\frac{4}{9}\right)^2 } = \sqrt{ \frac{16}{3} }. \end{align*}
    The point on the plane which is closest to \(P\) then has the position vector
    \begin{equation*} \mbf{q} = \vect{OP} - \operatorname{proj}_{\mathbf{n}}\mathbf{v} = \begin{bmatrix} 2\\ 3\\ 0 \end{bmatrix} - \frac{4}{9} \begin{bmatrix} 5\\ 1\\ 1 \end{bmatrix} = \frac{1}{9} \begin{bmatrix} -2 \\ 23\\ -4 \end{bmatrix}, \end{equation*}
    so that \(Q =(\frac{-2}{9},\frac{23}{9},\frac{-4}{9})\text{.}\)
  2. For the plane with general equation \(2x_1+x_2-x_3 =6\text{,}\) we can choose
    \begin{equation*} \mbf{n} = \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} \text{ and } A = (3, 0, 0). \end{equation*}
    We then compute
    \begin{equation*} \mbf{v} := \vect{OP}-\vect{OA} = \begin{bmatrix} 3\\ 1\\ -1 \end{bmatrix} - \begin{bmatrix} 3\\ 0\\ 0 \end{bmatrix} = \begin{bmatrix} 0\\ 1\\ -1 \end{bmatrix}, \end{equation*}
    so that
    \begin{equation*} \mbf{n} \cdot \mbf{v} = \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} \cdot \begin{bmatrix} 0\\ 1\\ -1 \end{bmatrix} = 2 \end{equation*}
    and
    \begin{equation*} \mbf{n} \cdot \mbf{n} = \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} \cdot \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} = 4+1+1 = 6. \end{equation*}
    We can now compute (using Hint 1.2.8.1)
    \begin{equation*} \operatorname{proj}_{\mathbf{n}}(\mathbf{v}) = \left( \frac{\mbf{n}\cdot \mbf{v}}{\mbf{n}\cdot \mbf{n}} \right) \mbf{n} = \frac{1}{3} \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} \end{equation*}
    and thus
    \begin{align*} \operatorname{dist} (\mathcal{P}, P) \amp = \norm{\operatorname{proj}_{\mathbf{n}}\mathbf{v}}\\ \amp = \sqrt{ \left(\frac{2}{3}\right)^2 + \left(\frac{1}{3}\right)^2 + \left(\frac{-1}{3}\right)^2 } = \sqrt{ \frac{2}{3} }. \end{align*}
    The point on the plane which is closest to \(P\) then has the position vector
    \begin{equation*} \mbf{q} = \vect{OP} - \operatorname{proj}_{\mathbf{n}}\mathbf{v} = \begin{bmatrix} 3\\ 1\\ -1 \end{bmatrix} - \frac{1}{3} \begin{bmatrix} 2\\ 1\\ -1 \end{bmatrix} = \frac{1}{3} \begin{bmatrix} 7\\ 2\\ -2 \end{bmatrix} \end{equation*}
    so that \(Q =(\frac{7}{3}, \frac{2}{3}, \frac{-2}{3})\text{.}\)
