The set \(S \) spans the set of all \(2 \times 2 \) matrices if and only if, for any \(2 \times 2 \) matrix \(A\text{,}\) there exist numbers \(x_{1}, x_{2}, x_{3}\) so that
\begin{equation*} x_{1} \begin{bmatrix} 1 \amp 2 \\ 2 \amp 1 \end{bmatrix} + x_{2}\begin{bmatrix} 2 \amp 1 \\ -1 \amp 2 \end{bmatrix} + x_{3} \begin{bmatrix} 0 \amp 1 \\ 1 \amp 2 \end{bmatrix} = A . \end{equation*}
If we write \(A = \begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix}\text{,}\) then this is equivalent to saying that there exists a solution to the following system of linear equations:
\begin{align*} x_{1} + 2x_{2} \amp = a\\ 2 x_{1} + x_{2} + x_{3} \amp= b\\ 2x_{1} - x_{2} + x_{3} \amp = c\\ x_{1} + 2x_{2} + 2x_{3} \amp = d \end{align*}
Let us transfer this into an augmented matrix and then row-reduce:
\begin{align*} \amp\matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 2 \amp 1 \amp 1 \amp b \\ 2 \amp -1 \amp 1 \amp c \\ 1 \amp 2 \amp 2 \amp d } \overset{-R_{1}+R_{4}}{\underset{-2R_{1}+R_{3}, -2R_{1}+R_{2}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -3 \amp 1 \amp b-2a \\ 0 \amp -5 \amp 1 \amp c-2a \\ 0 \amp 0 \amp 2 \amp d - a }\\ \amp\overset{\frac{1}{2}R_{4}}{\underset{-R_{4}'+R_{3}, -R_{4}'+R_{2}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -3 \amp 0 \amp b-2a -\frac{1}{2}(d - a) \\ 0 \amp -5 \amp 0 \amp c-2a - \frac{1}{2}(d - a) \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }\\ \amp\overset{5R_{2}}{\underset{-3R_{3}}{\longrightarrow}} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -15 \amp 0 \amp 5b-\frac{15}{2}a -\frac{5}{2}d \\ 0 \amp 15 \amp 0 \amp -3c+ \frac{9}{2}a + \frac{3}{2}d \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }\\ \amp\overset{R_{2}+R_{3}}{\longrightarrow} \matr{ccc|c} { 1 \amp 2 \amp 0 \amp a \\ 0 \amp -15 \amp 0 \amp 5b-\frac{15}{2}a -\frac{5}{2}d \\ 0 \amp 0 \amp 0 \amp (-3c+ \frac{9}{2}a + \frac{3}{2}d) + (5b-\frac{15}{2}a -\frac{5}{2}d) \\ 0 \amp 0 \amp 1 \amp \frac{1}{2}(d - a) }. \end{align*}
We see from the third row that the system is inconsistent if
\begin{equation*} -3c -3 a - d + 5b = \left(-3c+ \frac{9}{2}a + \frac{3}{2}d) + (5b-\frac{15}{2}a -\frac{5}{2}d\right) \neq 0. \end{equation*}
Since \(\begin{bmatrix} a \amp b \\ c \amp d \end{bmatrix} = \begin{bmatrix} 1 \amp 0 \\ 0 \amp 0 \end{bmatrix}\) is an example of a choice of numbers which makes the system inconsistent, we conclude that, no, \(S\) does not span the set of all \(2\times 2\) matrices.
