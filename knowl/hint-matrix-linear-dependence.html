Hint 3.2.3.1.
Matrices \(A_{1}, \ldots, A_{m}\) of the same size are called linearly dependent if there exist numbers \(c_{1},\ldots, c_{m}\text{,}\) not all of which are zero, so that
\begin{equation*} c_{1}A_{1}+c_{2}A_{2}+\ldots + c_{m}A_{m} = 0. \end{equation*}
Otherwise, they are called linearly independent.
