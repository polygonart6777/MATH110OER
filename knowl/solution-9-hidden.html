1. \begin{align*} (k+l)(\mathbf{u}+\mathbf{v}) \amp = (k+l)\mathbf{u}+ (k+l)\mathbf{v} \amp (\knowl{./knowl/axioms-vector-addition-distributivity-1.html}{\text{Distributivity 1 (1.1.8)}})\\ \amp = k\mathbf{u}+l\mathbf{u}+ k\mathbf{v}+l\mathbf{v} \amp (\knowl{./knowl/axioms-vector-addition-distributivity-2.html}{\text{Distributivity 2 (1.1.9)}})\\ \amp = k \mathbf{u} +k \mathbf{v} + l\mathbf{u} + l \mathbf{v} \amp (\knowl{./knowl/axioms-vector-addition-commutativity.html}{\text{Commutativity (1.1.3)}}) \end{align*}

    Notice that one could also first have used Distributivity 2 (1.1.9) and only then Distributivity 1 (1.1.8).

  2. First, we compute:

    \begin{align*} 0 \mathbf{u} \amp = (0 \cdot 2)\mathbf{u} \amp \text{ because } 0=0\cdot 2\\ \amp = 0 ( 2 \mathbf{u}) \amp (\knowl{./knowl/axioms-vector-addition-compatibility.html}{\text{Compatibility (1.1.7)}})\\ \amp = 0 (\mathbf{u}+\mathbf{u}) \amp \text{(Definition)}\\ \amp = 0\mathbf{u} + 0\mathbf{u} \amp (\knowl{./knowl/axioms-vector-addition-distributivity-2.html}{\text{Distributivity 2 (1.1.9)}}) \end{align*}

    To sum up, we have proved:

    \begin{equation*} 0 \mathbf{u} = 0\mathbf{u} + 0\mathbf{u} \end{equation*}

    If we add \(- 0\mathbf{u}\) on both sides, we get:

    \begin{equation*} 0 \mathbf{u} +(-0\mathbf{u}) = (0\mathbf{u} + 0\mathbf{u}) + (-0\mathbf{u}) \end{equation*}

    For the left-hand side, note that

    \begin{equation*} 0 \mathbf{u} +(-0\mathbf{u}) = \mbf{0} \end{equation*}

    by the axiom of the inverse element (1.1.5). For the right-hand side, we have

    \begin{align*} (0\mathbf{u} + 0\mathbf{u}) + (-0\mathbf{u}) \amp = 0\mathbf{u} + (0\mathbf{u} + (-0\mathbf{u})) \amp (\knowl{./knowl/axioms-vector-addition-associativity.html}{\text{Associativity (1.1.2)}})\\ \amp = 0\mathbf{u} + \mbf{0} \amp (\knowl{./knowl/axioms-vector-addition-inverses.html}{\text{Inverse elements (1.1.5)}})\\ \amp = 0\mathbf{u} \amp (\knowl{./knowl/axioms-vector-addition-neutral-elt.html}{\text{Neutral element (1.1.4)}}) \end{align*}

    All in all, we have shown

    \begin{equation*} \mbf{0} = 0 \mathbf{u} +(-0\mathbf{u}) = (0\mathbf{u} + 0\mathbf{u}) + (-0\mathbf{u}) = 0\mbf{u}, \end{equation*}

    as claimed.

  3. We have to show that \((-1)\mathbf{u}\) behaves like the inverse elements \(-\mathbf{u}\) whose existence is claimed in this axiom (1.1.5). In other words, it suffices to show that
    \begin{equation*} \mbf{u} + (-1)\mbf{u} = \mbf{0}. \end{equation*}
    We compute:
    \begin{align*} \mbf{u} + (-1)\mbf{u} \amp = 1\mbf{u} + (-1)\mbf{u} \amp (\knowl{./knowl/axioms-vector-addition-scalar-1.html}{\text{Identity of scalar multiplication (1.1.6)}})\\ \amp = (1+ (-1))\mbf{u} \amp (\knowl{./knowl/axioms-vector-addition-distributivity-2.html}{\text{Distributivity 2 (1.1.9)}})\\ \amp = 0\mbf{u} \amp\\ \amp = \mathbf{0} \amp (\knowl{./knowl/ex-sums-of-vecs3-item-neutral.html}{\text{Item b}}) \end{align*}

  4. First of all,

    \begin{align*} -(k\mathbf{u}) \amp = (-1)(k\mathbf{u}) \amp (\knowl{./knowl/ex-sums-of-vecs3-item-inverse.html}{\text{Item c}})\\ \amp = ((-1)k)\mathbf{u} \amp (\knowl{./knowl/axioms-vector-addition-compatibility.html}{\text{Compatibility (1.1.7)}}) \end{align*}

    We can now do two things: on the one hand, \((-1)k=-k\text{,}\) which proves that \(-(k\mathbf{u}) = -k \mbf{u}\text{.}\) On the other hand, \((-1)k = k(-1)\text{,}\) so that:

    \begin{align*} -(k\mathbf{u}) \amp = ((-1)k)\mathbf{u} \amp \text{ as argued above}\\ \amp = (k(-1))\mathbf{u} \amp \\ \amp = k((-1)\mathbf{u}) \amp (\knowl{./knowl/axioms-vector-addition-compatibility.html}{\text{Compatibility (1.1.7)}})\\ \amp = k(-\mathbf{u}) \amp (\knowl{./knowl/ex-sums-of-vecs3-item-inverse.html}{\text{Item c}}) \end{align*}
