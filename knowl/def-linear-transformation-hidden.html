(Source: [3], Definition 6.1)

A vector function \(T\colon \mathbb{R}^n \to \mathbb{R}^m\) is called linear transformation if it satisfies the following two conditions:
  1. \(T\) preserves addition: for all \(\mathbf{v}, \mathbf{u}\in\mathbb{R}^n\text{,}\) \(T(\mathbf{v}+ \mathbf{u})= T(\mathbf{v})+T(\mathbf{u})\text{,}\) and
  2. \(T\) preserves scalar multiplication: for all \(\mathbf{u}\in\mathbb{R}^n\) and \(k\in\mathbb{R}\text{,}\) \(T(k\mathbf{u})= kT(\mathbf{u})\text{.}\)
