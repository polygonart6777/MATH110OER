Item 3.4.1.e.ii.
Closed under scalar multiplication: We take an arbitrary vector \(\mathbf{u}\in V_{5}\) and any scalar \(\lambda\text{.}\) Since \(\mathbf{u}\in V_{5}\text{,}\) we know that \(\mathbf{u} = \begin{bmatrix} 0 \\ u_{2} \\ 0 \end{bmatrix}\) for some number \(u_{2}\text{,}\) so that
\begin{equation*} \lambda \mathbf{u} = \begin{bmatrix} 0 \\ \lambda \cdotu_{2} \\ 0 \end{bmatrix}. \end{equation*}
The first and third component of \(\lambda \mathbf{u}\) are equal to zero, so it is an element of \(V_{5}\text{.}\)
