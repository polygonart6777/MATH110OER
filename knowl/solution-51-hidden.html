We may start with the line \(3x+y =3.\) First notice that setting \(y = 0 \) gives \(x = 1. \) And setting \(x = 0 \) we get \(y = 3.\) The following graph is obtained by plotting these two solutions, \((1,0), (0,3) \) and connecting them to form a line.
Similarly, for the second equation we find the solutions \((1,0), (0,.5) \text{,}\) plot them on the same set of axes as the first line, and connect the two points to form a line.
The solution to the system of equations is the point of intersection of the lines, which we can graphically approximate to be \((1, 0) .\) We can check this solution by plugging it back into both equations.
\begin{gather*} 3(1) + 0 = 3\\ 1 - 0= 1. \end{gather*}
