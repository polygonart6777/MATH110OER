We require \(\mbf{a}\) to be parallel to \(\mbf{u}\text{,}\) i.e. \(\mbf{a} = c \mbf{u}\) for some real number \(c\) that is yet to be determined. Plugging that into \(\mathbf{v} = \mathbf{a}+\mathbf{b}\text{,}\) we get a definition of \(\mbf{b}\) that also depends on \(c\text{:}\) \(\mathbf{b} = \mathbf{v} - c \mbf{u}\text{.}\) As \(\mbf{b}\) needs to be orthogonal to \(\mbf{u}\text{,}\) we want \(\mbf{b}\cdot \mbf{u}=0\text{.}\) If we plug our tentative solution for \(\mbf{b}\) into this, we get
\begin{equation*} (\mathbf{v} - c \mbf{u})\cdot \mbf{u}=0 . \end{equation*}
Using the properties of the dot product, we arrive at
\begin{equation*} \mathbf{v}\cdot \mbf{u} = c \mbf{u}\cdot \mbf{u}, \text{ i.e. } c = \frac{\mathbf{v}\cdot \mbf{u}}{\mbf{u}\cdot \mbf{u}} \text{ using that }\mbf{u}\neq \mbf{0} . \end{equation*}
We can now compute both sides and get a value for \(c\text{:}\)
\begin{align*} \mathbf{v}\cdot \mbf{u} \amp = \begin{bmatrix} 1 \\ -1 \\ 2 \end{bmatrix} \cdot \begin{bmatrix} 3 \\ 2\\ -5 \end{bmatrix} = 1\cdot 3 + (-1)\cdot 2 + 2\cdot (-5) = -9,\\ \mbf{u}\cdot \mbf{u} \amp = \begin{bmatrix} 3 \\ 2\\ -5 \end{bmatrix} \cdot \begin{bmatrix} 3 \\ 2\\ -5 \end{bmatrix} = 3^2 + 2^2 + (-5)^2 = 38, \text{ so that } \end{align*}
\begin{equation*} c = \frac{-9}{38}. \end{equation*}
We can now solve for our vectors \(\mbf{a}\) and \(\mbf{b}\text{:}\)
\begin{gather*} \mbf{a} = c \mbf{u} = \frac{-9}{38}\begin{bmatrix} 3 \\ 2\\ -5 \end{bmatrix} = \begin{bmatrix} \frac{-27}{38} \\ \frac{-18}{38}\\ \frac{45}{38}\end{bmatrix} \\ \mathbf{b} = \mathbf{v} - \mbf{a} = \begin{bmatrix} 1 \\ -1 \\ 2 \end{bmatrix} - \begin{bmatrix} \frac{-27}{38} \\ \frac{-18}{38}\\ \frac{45}{38}\end{bmatrix} = \begin{bmatrix} \frac{65}{38} \\ \frac{-20}{38}\\ \frac{31}{38}\end{bmatrix} . \end{gather*}
Let us do a sanity check: Is our \(\mbf{b}\) indeed orthogonal to \(\mbf{u}\text{?}\) We compute:
\begin{align*} \mbf{b}\cdot \mbf{u} \amp = \begin{bmatrix} \frac{65}{38} \\ \frac{-20}{38}\\ \frac{31}{38}\end{bmatrix} \cdot \begin{bmatrix} 3 \\ 2\\ -5 \end{bmatrix} = \frac{65}{38} \cdot 3 + \frac{-20}{38}\cdot 2 + \frac{31}{38}\cdot (-5)\\ \amp = \frac{195-40-155}{38} = 0, \end{align*}
so yes!
