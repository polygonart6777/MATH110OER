Use determinants to find which real values of c make each of the following matrices invertible.
  1. \(\displaystyle \begin{bmatrix} 1 \amp 0 \amp 3 \\ 3 \amp -4 \amp c \\ 2 \amp 5 \amp 8 \end{bmatrix} \)
  2. \(\displaystyle \begin{bmatrix} 0 \amp c \amp -c \\ -1 \amp 2 \amp 1 \\ c \amp -c \amp c \end{bmatrix} \)
  3. \(\displaystyle \begin{bmatrix} c \amp 1 \amp 0 \\ 0 \amp 2 \amp c \\ -1 \amp c \amp 5 \end{bmatrix} \)
  4. \(\displaystyle \begin{bmatrix} 4 \amp c \amp 3 \\ c \amp 2 \amp c \\ 5 \amp c \amp 4 \end{bmatrix}\)
  5. \(\displaystyle \begin{bmatrix} 1 \amp 2 \amp -1 \\ 0 \amp -1 \amp c \\ 2 \amp c \amp 1 \end{bmatrix} \)
  6. \(\displaystyle \begin{bmatrix} 1 \amp c \amp -1 \\ c \amp 1 \amp 1 \\ 0 \amp 1 \amp c \end{bmatrix} \)
