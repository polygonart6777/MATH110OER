Exercises Exercises

1.

(Source: [4] in Section 1.2, Exercise 1.2.1.)

Which of the following matrices are in reduced row-echelon form? Which are in row-echelon form?
  1. \(\begin{bmatrix} 1\amp -1 \amp 2\\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
  2. \(\begin{bmatrix} 2\amp 1 \amp -1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\)
  3. \(\begin{bmatrix}1\amp -2 \amp 3 \amp 5 \\ 0 \amp 0 \amp 0 \amp 1 \end{bmatrix}\)
  4. \(\begin{bmatrix} 1\amp 0 \amp 0 \amp 3 \amp 1 \\ 0 \amp 0 \amp 0 \amp 1 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{bmatrix}\)
  5. \(\begin{bmatrix} 1 \amp 1 \\ 0 \amp 1\end{bmatrix}\)
  6. \(\begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
Hint 1

(Source: [4] Section 1.2, Definition 1.12)

A matrix is in row-echelon form (REF) if
  1. all nonzero rows are above any rows of zeros,
  2. each leading entry of a row is in a column to the right of the leading entries of any row above it, and
  3. each leading entry of a row is equal to \(1\text{.}\)

(Note that some sources do not assume that the leading entries are \(1\) for the REF.)

Hint 2

(Source: [4] Section 1.2, Definition 1.13)

A matrix is in reduced row-echelon form (RREF) if
  1. it is in REF and
  2. all entries in a column above and below a leading entry are zero.
Answer
  1. Neither in REF nor in RREF.
  2. Neither in REF nor in RREF.
  3. In REF, but not in RREF.
  4. In REF, but not in RREF.
  5. In REF, but not in RREF.
  6. Neither in REF nor in RREF.
Solution
  1. The matrix is not in REF: The zero row is not at the bottom.
  2. The matrix is in not in REF: The leading entry of the first row is not \(1\text{.}\) In particular, the matrix is not in RREF.
  3. The matrix is in REF, but not in RREF: The last column, which contains a leading \(1\) in the second row, has a non-zero entry (\(5\)).
  4. The matrix is in REF, but not in RREF: The fourth column, which contains a leading \(1\) in the second row, has a non-zero entry (\(3\)).
  5. The matrix is in REF, but not in RREF: The second column, which contains a leading \(1\) in the second row, has a non-zero entry (\(1\)).
  6. The matrix is not in REF: The leading entry of the second row (\(1\) in the \((2, 3)\)-entry) is not strictly to the right of the leading entry of the first row (\(1\) in the \((1, 3)\)-entry). In particular, the matrix is not in RREF.

2.

(Source: [2], Chapter 3, Section 1 (Gauss-Jordan Reduction) Problem 4.)

Give two distinct echelon form versions of this matrix.
\begin{equation*} \begin{bmatrix} 2 \amp 1 \amp 1 \amp 3 \\ 6 \amp 4 \amp 1 \amp 2 \\ 1 \amp 5 \amp 1 \amp 5 \end{bmatrix} \end{equation*}
Hint
See Hint 2.2.1.1 for the definition of the row-echelon form.
Answer
\begin{equation*} \begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp -2 \amp -7 \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix} \quad\text{ and }\quad \begin{bmatrix} 1 \amp 0 \amp 0 \amp \frac{-10}{19} \\ 0 \amp 1 \amp 0 \amp \frac{7}{19} \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix} \end{equation*}
(There are many other solutions, since the REF is not unique.)
Solution
We start by row reducing from top to botteom:
\begin{align*} \begin{bmatrix} 2 \amp 1 \amp 1 \amp 3 \\ 6 \amp 4 \amp 1 \amp 2 \\ 1 \amp 5 \amp 1 \amp 5 \end{bmatrix} \overset{-3R_{1}+R_{2}}{\underset{-R_{1}+2R_{3}}{\longrightarrow}} \amp\begin{bmatrix} 2 \amp 1 \amp 1 \amp 3 \\ 0 \amp 1 \amp -2 \amp -7 \\ 0 \amp 9 \amp 1 \amp 7 \end{bmatrix}\\ \overset{-9R_{2} + R_{3}}{\underset{\frac{1}{2}R_{1}}{\longrightarrow}} \amp\begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp -2 \amp -7 \\ 0 \amp 0 \amp 19 \amp 70 \end{bmatrix}\\ \overset{\frac{1}{19}R_{3}}{\longrightarrow} \amp\begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp -2 \amp -7 \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix}. \end{align*}
We have arrived at a matrix in REF. (We recall that some books do not require the leading entries to be \(1\) for a matrix to be in REF, in which case the second to last matrix in the above computation would also be considered to be in REF.) Since we are supposed to give two answers, we continue:
\begin{align*} \begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp -2 \amp -7 \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix} \overset{2R_{3}+R_{2}}{\longrightarrow} \begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp 0 \amp \frac{7}{19} \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix}. \end{align*}
Since we were not asked for a matrix in RREF, we could stop here and have our second answer. But instead, we might as well continue to manipulate our matrix to arrive at RREF:
\begin{align*} \begin{bmatrix} 1 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 1 \amp 0 \amp \frac{7}{19} \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix} \overset{-\frac{1}{2}R_{2}+R_{1}}{\longrightarrow} \amp\begin{bmatrix} 1 \amp 0 \amp \frac{1}{2} \amp \frac{25}{19} \\ 0 \amp 1 \amp 0 \amp \frac{7}{19} \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix}\\ \overset{-\frac{1}{2}R_{3} + R_{1}}{\longrightarrow} \amp\begin{bmatrix} 1 \amp 0 \amp 0 \amp \frac{-10}{19} \\ 0 \amp 1 \amp 0 \amp \frac{7}{19} \\ 0 \amp 0 \amp 1 \amp \frac{70}{19} \end{bmatrix}. \end{align*}

3.

(Source: [1], Section 1.2, Exercise 1.2.20.)

Row reduce the following matrix to obtain a row-echelon form. Then continue to obtain the reduced row-echelon form.
\begin{equation*} \begin{bmatrix} 2 \amp -1 \amp 3 \amp -1 \\ 1 \amp 0 \amp 2 \amp 1 \\ 1 \amp -1 \amp 1 \amp -2 \end{bmatrix}. \end{equation*}
Hint
See Hint 2.2.1.1 for the definition of REF and Hint 2.2.1.2 for the definition of the RREF.
Answer
There are many answers for the REF, for example
\begin{equation*} \begin{bmatrix} 1 \amp -\frac{1}{2} \amp \frac{3}{2} \amp -\frac{1}{2} \\ 0 \amp 1 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}. \end{equation*}
As always, the RREF is unique, and in this case, it is given by
\begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 2 \amp -1 \\ 0 \amp 1 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}. \end{equation*}
Solution
\begin{align*} \begin{bmatrix} 2 \amp -1 \amp 3 \amp -1 \\ 1 \amp 0 \amp 2 \amp 1 \\ 1 \amp -1 \amp 1 \amp -2 \end{bmatrix} \overset{-\frac{1}{2}R_{1} + R_{2}}{\underset{-\frac{1}{2}R_{1} + R_{3}}{\longrightarrow}} \amp \begin{bmatrix} 2 \amp -1 \amp 3 \amp -1 \\ 0 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp -\frac{1}{2} \amp -\frac{1}{2} \amp -\frac{3}{2} \end{bmatrix}\\ \overset{\frac{1}{2}R_{1}}{\underset{R_{2}+R_{3}}{\longrightarrow}} \amp \begin{bmatrix} 1 \amp -\frac{1}{2} \amp \frac{3}{2} \amp -\frac{1}{2} \\ 0 \amp \frac{1}{2} \amp \frac{1}{2} \amp \frac{3}{2} \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\\ \overset{2R_{2}}{\longrightarrow} \amp \begin{bmatrix} 1 \amp -\frac{1}{2} \amp \frac{3}{2} \amp -\frac{1}{2} \\ 0 \amp 1 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}. \end{align*}
We have arrived at a matrix in REF. To get it into RREF, we need to make sure that any column with a leading \(1\) has no other non-zero entries, so:
\begin{equation*} \begin{bmatrix} 1 \amp -\frac{1}{2} \amp \frac{3}{2} \amp -\frac{1}{2} \\ 0 \amp 1 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}. \overset{\frac{1}{2}R_{2} + R_{1}}{\longrightarrow} \begin{bmatrix} 1 \amp 0 \amp 2 \amp -1 \\ 0 \amp 1 \amp 1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}. \end{equation*}

5.

(Source: [5] in the Section for Row-reduced Echelon Form, Exercise RREF.T10.)

Prove that each of the three row operations is reversible. More precisely, if the matrix \(B\) is obtained from \(A\) by application of a single row operation, show that there is a single row operation that will transform B back into \(A\text{.}\)
Hint

(Adapted from [5], http://linear.ups.edu/fcla/section-RREF.html)

The three row operations are given by:
  1. Swap the locations of row \(i\) with row \(j\text{:}\) \(R_{i} \leftrightarrow R_{j}\text{.}\)
  2. Multiply each entry of a single row \(i\) by a nonzero quantity \(\alpha\text{:}\) \(\alpha R_{i}\text{.}\)
  3. Multiply each entry of one row \(R_{i}\) by some quantity \(\alpha\text{,}\) and add these values to the entries in the same columns of a second row \(R_{j}\text{.}\) Leave the first row the same after this operation (i.e. \(R_{i}\) remains unchanged), but replace the second row by the new values (i.e. row \(j\) is new): \(\alpha R_{i} + R_{j}\text{.}\)
Solution
First, assume that \(B \overset{R_{i}\leftrightarrow R_{j}}{\longrightarrow} A\text{.}\) Then \(A \overset{R_{j}\leftrightarrow R_{i}}{\longrightarrow} B\text{,}\) so the first row operation is invertible. To be more precise, the first row operation is its own inverse. Next, assume that \(B \overset{\alpha R_{i}}{\longrightarrow} A\text{.}\) For this to be a legitimate row operation, we must have \(\alpha\neq 0\text{.}\) In particular, \(\frac{1}{\alpha}\) exists and \(A \overset{\frac{1}{\alpha} R_{i}}{\longrightarrow} B\) is also a legitimate row operation. Thus, the row operation of multiplication by \(\alpha\) is invertible with inverse the row operation of multiplication by \(\frac{1}{\alpha}\text{.}\) Lastly, assume that \(B \overset{\alpha R_{i} + R_{j}}{\longrightarrow} A\text{.}\) If \(\alpha = 0 \) or row \(i\) in \(B\) is a zero-row, then the operation does nothing and \(B=A\text{.}\) If \(\alpha\neq 0\) and row \(i\) in \(B\) is not a zero-row, then \(A\) differs from \(B\) exactly in row \(j\text{;}\) in particular, row \(i\) is the same in \(A\) and \(B\text{.}\) We see from this that \(A \overset{-\frac{1}{\alpha} R_{i} + R_{j}}{\longrightarrow} B\) inverts the row operation.

6.

(Source: [4] in Section 1.2.)

Find the rank of each of the matrices in Problem 1, that is:
  1. \(\begin{bmatrix} 1\amp -1 \amp 2\\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
  2. \(\begin{bmatrix} 2\amp 1 \amp -1 \amp 3 \\ 0 \amp 0 \amp 0 \amp 0 \end{bmatrix}\)
  3. \(\begin{bmatrix}1\amp -2 \amp 3 \amp 5 \\ 0 \amp 0 \amp 0 \amp 1 \end{bmatrix}\)
  4. \(\begin{bmatrix} 1\amp 0 \amp 0 \amp 3 \amp 1 \\ 0 \amp 0 \amp 0 \amp 1 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \end{bmatrix}\)
  5. \(\begin{bmatrix} 1 \amp 1 \\ 0 \amp 1\end{bmatrix}\)
  6. \(\begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
Hint

(Source: [4] Definition 1.4 in Section 1.2.)

The rank of a matrix \(A\) is the number of leading \(1\)s in any row-echelon matrix to which \(A\) can be carried by row operations. (For the definition of the row-echelon form, see Hint 2.2.1.1.)
Answer
  1. rank \(2\)
  2. rank \(1\)
  3. rank \(2\)
  4. rank \(3\)
  5. rank \(2\)
  6. rank \(1\)
Solution
  1. By exchanging row \(2\) and \(3\text{,}\) we arrive at a matrix in row-echelon form with two leading \(1\)s. Thus, the rank of the matrix is \(2\text{.}\)
  2. By dividing the first row by \(2\text{,}\) we arrive at a matrix in row-echelon form with one leading \(1\text{.}\) Thus, the rank of the matrix is \(1\text{.}\)
  3. Since the matrix is in REF, we just need to count the leading \(1\)s: the rank of the matrix is \(2\text{.}\)
  4. Since the matrix is in REF, we just need to count the leading \(1\)s: the rank of the matrix is \(3\text{.}\)
  5. Since the matrix is in REF, we just need to count the leading \(1\)s: the rank of the matrix is \(2\text{.}\)
  6. To bring the matrix in REF, we substract the first row from the second and from the third row. The resulting matrix is
    \begin{equation*} \begin{bmatrix} 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}, \end{equation*}
    which has one leading \(1\) and hence rank \(1\text{.}\)

16.

(Source: [1], Chapter 1, Exercise 1.2.3)

Do the three lines \(x+2y=1\text{,}\) \(2x-y=1\text{,}\) and \(4x+3y=3\) in \(\mathbb{R}^2\) have a common point of intersection? If so, find the point. If not, explain why they don't have a common point of intersection.
Hint
If \((x, y)\) is a point of intersection of the three lines, then \((x, y)\) must satisfy the equations of all three lines.
Answer
The three lines intersect at \(\left(\frac{3}{5}, \frac{1}{5}\right)\text{.}\)
Solution
To find out if there is a common point of intersection we want to find out if the three given equations have a simultaneous solution. We therefore turn the equations into an augmented matrix and row reduce:
\begin{equation*} \matr{rr|r}{1\amp 2\amp 1\\2\amp -1\amp 1\\4\amp 3\amp 3} \to \matr{rr|r}{1\amp 2\amp 1 \\ 0 \amp -5 \amp -1 \\ 0 \amp -5 \amp -1} \to \matr{rr|r}{1 \amp 0 \amp \frac{3}{5} \\ 0 \amp 1 \amp \frac{1}{5} \\ 0 \amp 0 \amp 0} \end{equation*}
The row reduced form tells us that there is a solution to this system, and that the only solution is \(x=\frac{3}{5}, y = \frac{1}{5}\text{.}\) Therefore the three given lines do have a common point of intersection, and that point is \(\left(\frac{3}{5}, \frac{1}{5}\right)\text{.}\)
